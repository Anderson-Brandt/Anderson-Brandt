## 🎮 Anderson Brandt

**`Desenvolvedor de Jogos`** **`Game Designer`**

Desenvolvedor de jogos com experiência em Unity, especializado na criação de jogos para PC e mobile. Fundador do AdInfinitum Studios, publiquei títulos no Itchi.Io e na Steam, onde atuei no desenvolvimento de mecânicas, game design, jogabilidades, gestão de projeto, entre outras funções. Possuo formação em Game Design, Jogos Digitais, Inteligência Artificial e Gerenciamento de Projetos de TI, trazendo uma abordagem técnica e estratégica para o desenvolvimento de jogos. Busco oportunidades para colaborar com estúdios e contribuir para o crescimento da indústria de games no Brasil. 🕹️🎯❤️
